# CPT.Recent.Entries
Display a list of the most recent "Custom Post Type" entries in the wordpress widgets.

------------
## Features
+ Widget title
+ Show all "Post Types" in installed Wordpress
+ Limit the number of entries to display
+ Translation reade
    + IR_FA
